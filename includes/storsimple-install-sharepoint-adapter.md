#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>SharePoint için StorSimple bağdaştırıcısını yüklemek için
1. Yükleyiciyi, SharePoint Merkezi Yönetim Web uygulamasını çalıştırmak üzere yapılandırılmış Web ön ucu (WFE) sunucusuna kopyalayın. 
2. WFE sunucusunda oturum açmak için yönetici ayrıcalıklarına sahip bir hesap kullanın.
3. Yükleyiciye çift tıklayın. SharePoint için StorSimple Bağdaştırıcısı Kurulum Sihirbazı başlar. Yüklemeyi başlatmak için **İleri** ' ye tıklayın.
   
    ![StorSimple Bağdaştırıcısı kurulum başlangıç sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. SharePoint için StorSimple Bağdaştırıcısı kurulum yapılandırması sayfasında, bir yükleme konumu seçin, VERI 0 ağ arabiriminin IP adresini StorSimple cihazınıza yazın ve ardından **İleri**' ye tıklayın. 
   
    ![StorSimple Bağdaştırıcısı kurulum yapılandırma sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Kurulum onayı sayfasında, **yükleme**' ye tıklayın.
   
    ![StorSimple Bağdaştırıcısı kurulum onay sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kurulum sihirbazını kapatmak için **son** ' a tıklayın.
   
    ![StorSimple Bağdaştırıcısı Kurulum tamamlandı sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. SharePoint Merkezi Yönetim sayfasını açın. SharePoint için StorSimple Bağdaştırıcısı bağlantılarını içeren bir StorSimple yapılandırma grubu görmeniz gerekir.
8. Sonraki adıma gidin: KÇY 'yi [yapılandırma](#configure-rbs).
