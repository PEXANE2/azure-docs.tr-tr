# <a name="pricing"></a>Fiyatlandırma
Cloud Shell ve Cloud Shell PowerShell 'deki Bash aşağıdaki bilgilere tabidir.

## <a name="compute-cost"></a>İşlem maliyeti
Azure Cloud Shell, Azure tarafından ücretsiz olarak sunulan bir makinede çalışır, ancak bir Azure dosya paylaşımının kullanmasını gerektirir. 

## <a name="storage-cost"></a>Depolama maliyeti
Cloud Shell, oturumlara yeni veya mevcut bir Azure dosyaları paylaşımının bağlanmasını gerektirir. Depolama, düzenli maliyetler doğurur.

[Azure dosyaları maliyetleriyle ilgili ayrıntılar için buraya](https://azure.microsoft.com/pricing/details/storage/files/)bakın.
